# 🦞 CloudClaw - 个人A股量化交易软件

小而精的个人量化交易软件，集成主流高胜率策略、技术分析和回测功能。

## 功能特性

### 四大交易策略
| 策略 | 类型 | 核心思想 | 适合行情 |
|------|------|---------|---------|
| 龙头战法 | 短线追涨 | 追踪涨停连板龙头股 | 强势市场 |
| 麻雀战法 | 高频短线 | 积小胜为大胜，严格止损 | 震荡市 |
| 海龟战法 | 趋势跟踪 | 突破入场，让利润奔跑 | 趋势市 |
| 价值投资 | 长线持有 | 寻找低估优质公司 | 熊市布局 |

每种策略覆盖完整的交易流程：选股 → 建仓 → 加仓 → 减持 → 清仓 → 止盈止损

## 安装

```bash
pip install -r requirements.txt
python main.py
```
