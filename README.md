## 历史特性 (v2.2.0)

### 评分驱动股票池层级系统

基于评分阈值的动态四层结构，池子大小随市场好坏自适应：

| 层级 | 评分 | 角色 | 维护频率 |
|------|------|------|---------|
| 🎯 **精选层 (Focus)** | ≥ 80 | 可直接候选开仓 | 每日评估 |
| 👀 **观察层 (Watch)** | 60-80 | 持续跟踪，等待信号 | 每日评估 |
| 📋 **备选层 (Broad)** | 40-60 | 潜力储备，待晋级 | 每周评估 |
| ❌ **淘汰 (Eliminated)** | < 40 | 自动标记移除 | 即时触发 |

**淘汰规则**：
- 评分跌破 30 → 自动淘汰
- 累计跌幅 > 8% → 硬止损淘汰
- 入池超 20 日未晋级 → 时效淘汰

**维护机制**：各策略独立频率（龙头每日 → 麻雀每2天 → 海龟每周 → 价值每2周），淘汰后自动从市场补入高评分新标的。

**仪表盘增强**：层级统计卡片 + 三维筛选按钮（全部/精选/观察/备选），标的按层级→评分排序显示。

### 热点板块排行

实时统计全市场涨停股所属板块，按涨停数量降序排列，展示板块 5日涨幅 Top10。

### 股票池层级管理 (v2.2.0)

评分驱动的三层动态池，配合自动淘汰和补入机制：

```bash
云侠> pool eval               # 评估所有策略池层级
云侠> pool eval dragon        # 评估龙头战法池
云侠> pool maintain           # 完整维护（评估+淘汰+补入）
云侠> pool tiers              # 查看各池层级统计
```

每种策略独立评分维度：
- 龙头战法：连板天数 + 量比 + 换手率 + 封板 + 趋势加速
- 麻雀战法：均线多头 + KDJ金叉 + 回调 + RSI + 量价配合
- 海龟战法：突破强度 + 趋势 + ATR波动率 + 流动性 + 加仓空间
- 价值投资：估值分位 + PE/PB + ROE + 股息率 + 市值 + 负债率
- 布林带回归：%B位置 + RSI + 波动率 + 成交量
- 网格交易：MA60偏离度 + RSI + 波动率 + 区间宽度
- 均线交叉：MA10/MA30金叉 + 量比 + 趋势强度
- 量价突破：20日高点突破 + 量比 + RSI + 趋势确认
- 趋势加速：均线多头排列 + 回调深度 + RSI + 动量

### 回测与赛马

- `backtest` - AKQuant 单策略高性能回测
- `compare` - 多策略对比（夏普/收益/回撤/胜率）
- `race` - 多策略赛马模拟盘（每日追踪排名）
