# CloudKnight - 云侠量化交易系统 v2.2.0

云侠量化交易系统，基于 **AKQuant** 高性能事件驱动回测引擎。
集成四大主流高胜率策略、技术分析、多策略赛马和 **A股时间周期实时交易引擎**。

## 最新特性 (v2.2.0)

### 评分驱动股票池层级系统

基于评分阈值的动态四层结构，池子大小随市场好坏自适应：

| 层级 | 评分 | 角色 | 维护频率 |
|------|------|------|---------|
| 🎯 **精选层 (Focus)** | ≥ 80 | 可直接候选开仓 | 每日评估 |
| 👀 **观察层 (Watch)** | 60-80 | 持续跟踪，等待信号 | 每日评估 |
| 📋 **备选层 (Broad)** | 40-60 | 潜力储备，待晋级 | 每周评估 |
| ❌ **淘汰 (Eliminated)** | < 40 | 自动标记移除 | 即时触发 |

**淘汰规则**：
- 评分跌破 30 → 自动淘汰
- 累计跌幅 > 8% → 硬止损淘汰
- 入池超 20 日未晋级 → 时效淘汰

**维护机制**：各策略独立频率（龙头每日 → 麻雀每2天 → 海龟每周 → 价值每2周），淘汰后自动从市场补入高评分新标的。

**仪表盘增强**：层级统计卡片 + 三维筛选按钮（全部/精选/观察/备选），标的按层级→评分排序显示。

### 热点板块排行

实时统计全市场涨停股所属板块，按涨停数量降序排列，展示板块 5日涨幅 Top10。

## 引擎升级

v2.2.0 核心回测引擎 [AKQuant](https://github.com/akfamily/akquant)：
- **Rust 内核**：20x 于传统 Python 框架的回测性能

- **多线程并行**：四种策略独立线程扫描信号

## 功能特性

### A股实时交易引擎

完整覆盖A股交易日的全天时间节点：

| 时间 | 阶段 | 引擎行为 |
|------|------|---------|
| 08:30 | 盘前准备 | 大盘/板块分析、市场情绪评估、股票池信号扫描 |
| 09:15 | 集合竞价 | 监控竞价走势 |
| 09:26 | 竞价结果 | 竞价量价分析、开盘交易决策 |
| 09:30 | 早盘开盘 | 分时走势跟踪、多策略信号扫描 |
| 11:30 | 午间休市 | 15分钟K线技术分析、制定下午交易计划 |
| 13:00 | 午盘开盘 | 继续分时跟踪、执行买卖点信号 |
| 15:00 | 收盘 | 盘后全市场选股、制定次日交易计划 |

**非交易日**：自动执行全策略股票池筛选

```bash
python main.py live           # 启动实时交易引擎（Ctrl+C 停止）
```

### 四大交易策略

| 策略 | 类型 | 核心思想 | 适合行情 |
|------|------|---------|---------|
| 龙头战法 | 短线追涨 | 追踪涨停连板龙头股 | 强势市场 |
| 麻雀战法 | 高频短线 | 积小胜为大胜，严格止损 | 震荡市 |
| 海龟战法 | 趋势跟踪 | 突破入场，金字塔加仓 | 趋势市 |
| 价值投资 | 长线持有 | 寻找低估优质公司 | 熊市布局 |

每种策略均实现 `CloudKnightStrategy → akquant.Strategy`，覆盖完整交易流程：
选股 → 建仓 → 加仓 → 减持 → 清仓 → 止盈止损

### 股票池层级管理 (v2.2.0)

评分驱动的三层动态池，配合自动淘汰和补入机制：

```bash
云侠> pool eval               # 评估所有策略池层级
云侠> pool eval dragon        # 评估龙头战法池
云侠> pool maintain           # 完整维护（评估+淘汰+补入）
云侠> pool tiers              # 查看各池层级统计
```

每种策略独立评分维度：
- 龙头战法：连板天数 + 量比 + 换手率 + 封板 + 趋势加速
- 麻雀战法：均线多头 + KDJ金叉 + 回调 + RSI + 量价配合
- 海龟战法：突破强度 + 趋势 + ATR波动率 + 流动性 + 加仓空间
- 价值投资：估值分位 + PE/PB + ROE + 股息率 + 市值 + 负债率

### 回测与赛马

- `backtest` - AKQuant 单策略高性能回测
- `compare` - 多策略对比（夏普/收益/回撤/胜率）
- `race` - 多策略赛马模拟盘（每日追踪排名）
